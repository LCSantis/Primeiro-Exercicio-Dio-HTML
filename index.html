  • Objetivo

    O objetivo desse desafio é escrever um documento HTML utilizando as tags aprendidas, tendo como conteúdo o material visto até o momento.

  • Primeiros computadores

    De início vale pensar no objetivo dos computadores. Eles servem para computar, em outras palavras, fazer cálculos matemáticos que, se fossem feitos a mão, levariam muito tempo para se chegar a um resultado.

    A primeira geração de computadores, na linha dos modernos, surgem na década de 40. Neste ponto temos Alan Turing, em meio a Segunda Guerra Mundial, desenvolvendo uma máquina, a válvulas, para quebrar códigos Nazistas. Códigos produzidos por uma máquina chamada ENIGMA, mecânica.

    Após esse evento os computadores foram evoluindo passando pelo Mark I, eletromecânico, por Eniac, sendo esse o primeiro computador eletrônico, chegando até a criação dos transistores gerando assim os computadores que utilizamos nos dias de hoje

  • Client x Server

    Client-side

    Client-side ou lado cliente, se refere a tudo o que roda na máquina de quem está acessando o website. Quando digitamos o endereço de um website o fazemos por um navegador / browser, ele enviará uma solicitação para o servidor onde o site está hospedado, esse servidor responderá enviando os arquivos necessários, então o navegador interpreta o HTML, CSS, JavaScript e transforma no visual que enxergamos pela tela.

    HTML, CSS e JavaScript são tecnologias que rodam no lado cliente, permitindo assim que o cliente possa visualizar o website.

    Server-side

    Já no lado servidor temos outras tecnologias que vão rodar dentro dele. PHP, Python, Java e até mesmo o JavaScript, que pode rodar tanto no Client-side como no Server-side, são exemplos disso.

  • Tags HTML

    O documento HTML é escrito por meio de tags. Cada tag representa a forma como o navegador irá interpretar aquele pedaço do documento.

      Exemplo de tags:

    • <h1> - <h6> -> Referente ao peso do título, começando de h1 e indo até h6, sendo que h1 é o mais importante;
    • <a> -> Referente a ancoragem, faz um link com algum outro site ou seção do documento;
    • <p> -> Referente a parágrafo, diz que a seção é um parágrafo;
    • <mark> -> Referente a marcação de texto, como um marcador permanente;

    Existem várias tags, o site w3schools disponibiliza uma lista com todas elas, para conferir clique aqui.
